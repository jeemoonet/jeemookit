# Taste Skill 变体（上游可选）

来源：[Leonxlnx/taste-skill](https://github.com/Leonxlnx/taste-skill)。安装名即 `npx skills add` 的 `--skill` 参数。

| 安装名 | 适用场景 |
|--------|----------|
| `design-taste-frontend` | 默认（本 skill，v2） |
| `design-taste-frontend-v1` | 锁定 v1 行为 |
| `gpt-taste` | GPT/Codex：更强布局与 GSAP |
| `image-to-code` | 出图 → 分析 → 写代码 |
| `redesign-existing-projects` | 现有项目 UI 审计后改版 |
| `high-end-visual-design` | 柔和高端、大留白、spring 动效 |
| `minimalist-ui` | Notion/Linear 风编辑型 UI |
| `industrial-brutalist-ui` | 粗野主义、瑞士排版 |
| `full-output-enforcement` | 防止半成品/占位注释 |
| `stitch-design-taste` | Google Stitch 兼容 |
| `imagegen-frontend-web` | 仅生成网站参考图（不写代码） |
| `imagegen-frontend-mobile` | 仅生成移动端参考图 |
| `brandkit` | 仅生成品牌板 |

**选型**：通用前端从本 skill 开始；已定视觉方向再加 minimalist / brutalist / soft；出图流程用 image-to-code 或 imagegen-*。
